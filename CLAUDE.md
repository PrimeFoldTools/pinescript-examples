# PineScript Examples

Public educational PineScript v5 examples for TradingView.

## Structure
- `examples/`: 6 .pine files (ATR trailing stop, EMA crossover, multi-timeframe EMA, risk-reward overlay, session volume profile, trade session filter)

## Conventions
- All code must be valid PineScript v5, paste-ready for TradingView
- Educational only — no proprietary logic from pinescript-source
- Each .pine file is self-contained with no external dependencies

## Guardrails
- **Public repo.** No GERTHUMS logic, no proprietary indicators, no trading signals.
- MIT licensed — keep it clean.

## Quick Commands
- List examples: `ls examples/`
